SwiftDropship

GitHub Pages deployment for the SwiftDropship project.

This project is built with Flask and SQLite, so dynamic features (login, cart, checkout, admin, database) require a Python server runtime and cannot run directly on GitHub Pages.

For full app hosting, deploy the backend app from this repository to a Python host (Render, Railway, Fly.io, or VPS).
